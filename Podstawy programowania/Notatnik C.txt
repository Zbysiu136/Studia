Modyfikator
.liczba - dokładność po przecinku np .3f
h - W połączeniu ze specyfikatorem całkowitym oznacza wartość short int lub unsigned short int Przykłady: %hu”, ”%hx”
j - używany ze specyfikatorem konwersji liczb całkowitych oznacza wartość typu intmax_t lub uintmax_h (typy zdefiniowano w stdint.h) Przykład: ”%jd”
l - oznacza long int unsigned long int %ld
ll - long long int dla inta tylko %lld
L - long double dla flout %lf
z - wartość size_t typ zwracający wartość sizeof dla inta %zd
